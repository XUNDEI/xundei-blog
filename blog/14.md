安装setuptools、安装预编译好的wheel都不好使。想回退3.13吧，又麻烦。最终，在网上查了一下，发现原pygame库开发已经陷入停滞，现在可以使用``pygame-ce``

接下来简单说说，pygame-ce到底是什么？曾经的信仰pygame到底怎么了？（以下内容摘自AI）

>pygame-ce（pygame Community Edition，即pygame社区版） 是经典 pygame 库的一个分支（fork），由原 pygame 项目的核心开发人员创建。
>
>它诞生是因为原 pygame 项目在治理上遇到了难以克服的挑战，导致核心开发者无法继续在原项目上推进开发工作。于是他们创建了这个社区驱动的版本，旨在提供更频繁的版本发布、持续的bug修复和功能增强，以及更民主的治理模式。
>
>简单来说，pygame-ce 就是 pygame 的“精神续作”——它继承了原库的全部功能和API，同时由原班核心团队继续维护和进化。

目前，pygame-ce的维护与开发更活跃积极，功能也比pygame更丰富，且支持**python3.14**

想安装很简单

首先，最好先删除原来的pygame（貌似不删也可以，有说法是pip 会自动卸载旧版并替换为社区版，因为它们在 PyPI 上被标记为互斥的替代包）
```shell
pip uninstall pygame
```

然后，直接安装pygame-ce即可
```shell
pip install pygame-ce
```

安装完成就好，之后照常``import pygame``即可！之前pygame的教程也是适用于pygame-ce的，已有项目也可以无痛迁移
