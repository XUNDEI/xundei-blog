## python简介

在动手之前，不妨让我们了解一下python吧（如果不想看可以通过右边目录直接跳过此部分）

### Python 是什么？

Python 是一种解释型、面向对象、动态数据类型的高级程序设计语言。简单说，就是写出来的代码容易读懂，而且写起来很快。

### 为什么选 Python？

* 语法简洁：完成同样的事情，Python 代码量通常是 Java、C++ 的 1/5 甚至更少
* 上手快：初学者几天就能写实用的小程序
* 应用广：爬虫、网站、数据分析、AI、自动化脚本都能做
* 社区庞大：遇到问题，搜一下基本都有现成答案

### Python 的缺点？

* 运行速度相对 C/C++ 慢一些
* 手机端支持弱

### python适合？

* 就业。目前在AI大环境下python可以说是最吃香的语言了，除开AI，科学计算、数据分析等都很实用
* 初学者。python的语法简洁易理解，我推荐python作为你的第一门编程语言来理解一些概念，为其他语言的学习打下基础
* 解决日常问题。在一些需要计算实现算法的场景，python可以用很少的代码完成计算。亦或者写一些简单的脚本来解决一些小需求
* 编写简单网站。python也可以作为后端语言，但不适合处理高并发需求
* 开源开发者。python的社区庞大且充满活力

### python不适合？
* 追求性能的场景。python是一门高级编程语言，不接触内存等复杂概念，因此你无法手动管理内存而是由python自动管理，因此python的性能有瓶颈，同时python因为GIL（全局解释器锁）的存在，不能进行大型多线程 CPU 密集型任务（比如同时压缩 100 个超大文件）。python也无法胜任大型游戏
* 系统底层/嵌入式开发。python不能接触底层，像内核、驱动等无法编写
* 移动端开发。python的移动端支持弱
* 防逆向。python的源代码很容易被扒出来，像付费软件的授权验证逻辑就不可能用python写

好了，废话有点多，现在我们就开始安装python环境吧！

### PyCharm

由JetBrains出品，是专门为Python打造的IDE（集成开发环境），和vscode这种万能王不同，PyCharm是专精Python的。vscode像一把瑞士军刀，啥都能干，干啥都还行，想要啥自己加；PyCharm开箱即用，自带一整套工具，专为python而生。

优势？

* 开箱即用：装完就能写Python，不用装插件、不用配解释器，省心
* 功能强大：调试、重构、测试、版本控制、数据库工具全都有
* 智能的代码补全：针对Python深度优化，补全准确率比vscode高一点
* 专业级调试：断点、变量监视、表达式求值，做大项目时很爽
* 虚拟环境管理：帮你自动管理不同项目的依赖，省得以后头疼

虽然pycharm很特化，但是不像vscode没明显短板，pycharm的短板很明显

劣势？
* 吃内存：8GB内存的电脑跑起来会有点喘，16GB以上才舒服
* 启动慢，很重：第一次打开要等十几秒甚至更久，低配老电脑不建议上
* 专业版收费：免费版够用，但有些功能被阉割了，专业版授权不便宜，如果你不是靠python吃饭的话买专业版亏麻了

### Sublime Text

这是一款极简编辑器，性能极佳，也可以像python一样通过扩展插件来丰富自身功能。这曾经是最热门的python编辑器之一，但现在Sublime Text作为python开发主力舰的时代已经过去了，因此我们这里不作详细介绍，但不是不能用，你也可以试试。

### Vim

Vim是一款流⾏的编辑器，深受众多经验丰富的程序员的喜爱，因为编程过程中双⼿根本不⽤离开键盘，效率会大幅提升。燃鹅，Vim的上手难度很高，并不适合现在的你，如果你的确想尝尝，可以试试vscode的Vim插件，这个插件也可以实现手不离开键盘编程

### python IDLE

这是python自带的IDE，只要你安装了python就会有，打开就能用。有人会推荐这个，因为这个很简单易上手。但我个人**强烈不推荐**，这东西太简陋了，连个括号都不舍得补全，体验会很难受，而且其他的编辑器也不会比IDLE难到哪去

### 总结

python的编辑器很多很多，这里碍于篇幅只介绍了最热门常见的几个，如果你看花了眼，不知道选什么的话，那我个人的建议是**vscode**。Why？你前面不是说pycharm又智能开箱即用吗？为什么不比vscode好？

别急，首先就是pycharm的提示过于智能了，虽然对于专业python程序员能极大提高效率，但对于初学者而言，频繁出现的侵入式代码补全会干扰思考，也会影响初学者培养构造程序的思维。其二就是开箱即用，有人看好，认为能专注于编程而不是配置。但我认为对于初学者配置环境是不得不品鉴的一环，如果已经依赖IDE那么就像温室的花朵，终有一天面对其他编程语言的时候你还是要配置环境。而且pycharm的免费版阉割了一些功能，但在vscode全都是免费的，等你习惯了pycharm但受限于付费功能再想转vscode会遇到很多习惯问题。最好就是pycharm太重了，很多功能初学者都用不上

不过总的来说，pycharm还是非常优秀的一个python IDE，想用也是没有问题的

## 配置虚拟环境（可选）

先别跑，我知道这名字听起来像什么高大上的东西，但其实很好理解╮(╯▽╰)╭

想象一下，你同时做两个项目。项目A要用某个库的 1.0 版本，项目B要用某个库的 2.0 版本。如果全装到系统里，后装的会把前一个覆盖掉，两个项目必死一个。

虚拟环境就是给每个项目一个独立的小房间，A用A的版本，B用B的版本，互不干扰。

虽然对于初学者不重要，但我还是希望你能尽早意识到在以后虚拟环境的重要性，我不希望在以后你看见一个python项目后直接在主环境`pip install -r requirements.txt`然后一堆东西跟着炸〣(ºΔº)〣

其实这不难，pycharm默认配置虚拟环境，pycharm用户可以跳过这一部分，我们这里以vscode做演示

首先，需要在vscode安装一个插件`Python Environment Manager`，这个插件可以可视化管理虚拟环境。然后打开前文你在安装完vscode，你为日后python开发所创建的文件夹，记下路径，然后Win+R打开cmd，输入下面内容回车即可
```shell
cd 你刚刚记下的文件路径
```
如果你的文件夹在其他盘符（比如D盘），就按如下输入
```shell
D:
（回车）
cd 你刚刚记下的文件路径
```
按上述步骤在cmd中打开你的项目文件后，输入以下内容回车
```shell
python -m venv 你想要创建的虚拟环境名字
```
虚拟环境名字任意都可，但不建议中文

然后回到vscode，不出意外左边会多一个python图标，点开后能看见我们刚创建的虚拟环境，直接点击此虚拟环境右边的红圈部分激活虚拟环境

## pip换源

pip默认从国外服务器下载库，这通常会很慢，因此我们需要为其更换为国内镜像源

首先需要更新pip
```shell
python -m pip install --upgrade pip
```
如果网络不好无法更新时，为了解决先有鸡还是先有蛋的问题，可以先临时使用镜像源更新
```shell
python -m pip install -i https://mirrors.tuna.tsinghua.edu.cn/pypi/web/simple --upgrade pip
```
更新完后，输入以下命令来更换镜像源，这个给大家准备了几个，如果你想的话可以试试哪个更快

```shell
#清华源
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
```
```shell
#阿里云
pip config set global.index-url https://mirrors.aliyun.com/pypi/simple/
```
```shell
#中国科技大学
pip config set global.index-url https://pypi.mirrors.ustc.edu.cn/simple/
```
```shell
#华为云
pip config set global.index-url https://repo.huaweicloud.com/repository/pypi/simple/
```
```shell
#腾讯云
pip config set global.index-url https://mirrors.cloud.tencent.com/pypi/simple/
```
```shell
#豆瓣源，有人说已失效，我亲测目前会直接重定向到腾讯云，无论怎么样还是不建议使用
pip config set global.index-url https://pypi.doubanio.com/simple/
```

然后输入以下内容来验证是否生效
```shell
pip config list
```

比如说我我用清华源，输出应该如下
```text
C:\Users\xundei>pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple
Writing to C:\Users\xundei\AppData\Roaming\pip\pip.ini

C:\Users\xundei>pip config list
global.index-url='https://pypi.tuna.tsinghua.edu.cn/simple'
```
可以看到`global.index-url`的值就等于我们设置的清华源`https://pypi.tuna.tsinghua.edu.cn/simple`

如果你要特殊需要更换回官方源，请输入
```shell
pip config unset global.index-url
```
